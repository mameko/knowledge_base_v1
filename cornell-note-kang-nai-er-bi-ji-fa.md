# Cornell Note（康奈尔笔记法）

**Use Case：**

* 上课时做笔记
* 做读书笔记

**Why：**

其实，看了一些网上的文章，书里也多次提到。感觉挺有趣，也应该挺有应用的，所以实践。譬如，《学习高手》（TODO: 等做了笔记连这里）以前我们做笔记，就从头到尾，几乎是照搬。到了复习的时候也是那样不过脑子的死记硬背，这样效率低。忘了是谁推广的，不重要。

**How：**

1. 工具，纸笔就行了。
2. 传统做法是把一页切开，但其实只要你分好部分就行，不用一定都放同一页。重点是分开不同的部分。

![](https://images.gitbook.com/__img/dpr=2,width=760,onerror=redirect,format=auto,signature=1338080025/https%3A%2F%2Ffiles.gitbook.com%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%2FPopm7FL9xZgW9F04Jfkr%2Fuploads%2FS2zEGUFLXz7bZ5UQ2U7f%2Fimage.png%3Falt%3Dmedia%26token%3D76d24e22-a82b-43dd-a8b8-837bb0f4266f)

1. 传统的好像没有“主题“，但其实没啥关系。重点是怎么用
2. 首先，把平时流水账，上课听讲的，或者读书看到的，写到**笔记**区&#x57DF;_&#x52;ecord_
   1. 这一步是上课时候做的，还有读书摘抄重点时候做的
3. 然后，**要点**部分在其他地方&#x53EB;_&#x52;educ&#x65;_&#x548C;_Recite_，是在复习的时候，遮住笔记区域，帮助自己回忆用的。例如，提问关于那部分笔记的内容。
   1. 关于问什么，可以：what，why，how，影响到什么，影响是什么，意义是什么，未来会怎么发展，自己发挥哈
   2. 这一步是在课后快速回忆写下来的，以后复习用
   3. 最好在48小时内写完这部分
4. 最后，把自己的感想呀，总结呀，写到**总结**部&#x5206;_&#x52;eflec&#x74;_&#x548C;_Review_
   1. 这一步好像是复习了以后写的
   2. 因为要写感想和总结，所以是某种形式的输出，考验自己到底有没有掌握知识。估计用自己的话写
5. 其实还有一步，温故而知新，请结合**艾宾浩斯遗忘曲线**(TODO:补充页面）食用

&#x20;**不足之处：**

这种方法，会以时间方式来记录。有时候，分主题来记录（或者多记录一遍）。对知识存储有益
